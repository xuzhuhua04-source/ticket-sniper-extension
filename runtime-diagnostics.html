Buying FAQ

Commercial readiness
Can every module be bought separately?

Yes. Core plans set account limits, while paid modules and add-ons unlock product surfaces through server-side entitlements.

Is Dev mode included for everyone?

No. Dev mode is a paid Pro add-on and is included automatically only with Enterprise access.

What happens before billing is configured?

The UI clearly marks sandbox preview access. Production sales require Authorize.Net API credentials, item amounts, and signed webhook configuration.

How SIG9 packages map to the runtime engine.

ArchitectureRuntime ledger

AccessServer entitlements
