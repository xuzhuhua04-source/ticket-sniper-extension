Calendar Detection

Adjust only when the official portal's calendar is not detected correctly.

Leave blank to use the built-in detectors. This selector only reads visible calendar elements.

Optional but recommended. When set, visa calendar notifications are sent only when the official page DOM says this is the active consulate. Runtime diagnostics for other websites are not filtered by this.

Copy the exact reschedule-page URL from your official embassy-linked scheduling portal. Notification clicks open this page; the extension never selects or submits an appointment.

Supported official scheduling hosts *.usvisascheduling.com atlasauth.b2clogin.com ais.usvisa-info.com *.ustraveldocs.com cgifederal.secure.force.com

Companion App

Forward sanitized events to the local installable app.

Provider secrets remain in the companion app's `.env`. This token only authenticates the extension to your local app.

Operational limits

The extension does not refresh pages, submit forms, reserve appointments, bypass CAPTCHA or queues, access hidden endpoints, or store usernames and passwords. The official portal's rules and session limits always control.
